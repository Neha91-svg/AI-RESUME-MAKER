# ResumePro – AI-Powered Professional Resume Maker

![ResumePro Banner](https://images.unsplash.com/photo-1586281380349-632531db7ed4?auto=format&fit=crop&q=80&w=1200)

**ResumePro** is a modern, high-performance web application designed to help job seekers create professional, ATS-friendly resumes in minutes. Built with a focus on speed, aesthetics, and ease of use, it features a real-time preview engine and customizable templates.

---

## ✨ Key Features

- 🚀 **Lightning Fast Editor**: Build your resume in under 5 minutes with our intuitive interface.
- 🎯 **ATS Optimized**: Templates designed to pass through Applicant Tracking Systems effortlessly.
- 👁️ **Real-time Preview**: See your changes instantly as you type.
- 🪄 **Modern Aesthetics**: Premium glassmorphism design with smooth animations.
- 🖨️ **Export to PDF**: High-quality print and PDF export support.
- 📱 **Fully Responsive**: Create your resume on any device.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: [React 19](https://react.dev/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS with modern variables & Glassmorphism
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

### Backend
- **Server**: [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/)
- **Environment**: [Dotenv](https://www.npmjs.com/package/dotenv)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Neha91-svg/AI-RESUME-MAKER.git
   cd AI-RESUME-MAKER
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

3. **Setup Frontend**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

4. **Open in Browser**
   Navigate to `http://localhost:5173` to see the app in action!
